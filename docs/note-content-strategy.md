# note コンテンツ戦略

## 前提：noteは「今日」を読ませる設計

- プロフィールは時系列のみ。古い記事は沈む
- note内検索が弱く、他人のアカウント内を絞り込めない
- アーカイブを掘らせる設計ではない（ブログよりSNSに近い）

## 核心：同じメッセージをマイナーチェンジして繰り返す

一回書いて伝わると思わない。テレビCMと同じ原理。

### やり方

- **切り口を変える** -- 同じ核を別の入り口で書く
- **具体例を差し替える** -- 季節、時事ネタ、体験談を入れ替え
- **タイトルを変える** -- 検索・タイムライン上の見え方が変わる
- **届く層が変わる** -- 入り口が違えば別の人に刺さる

### 核は同じ、ガワを変える

| 核のメッセージ | 切り口A | 切り口B | 切り口C |
|---|---|---|---|
| 内装は下地が全て | 失敗事例から学ぶ | プロの道具紹介から入る | お客様の声から入る |
| リフォームの判断基準 | 季節ごとの注意点 | 予算別の選択肢 | 築年数で考える |

## 自サイトとの使い分け

- **note** = 流入口。新規に見つけてもらう場所
- **自サイト** = アーカイブ。体系的に読ませる場所
- noteから自サイトへ導線を貼る

## 埋もれ対策（やらないよりマシ程度）

1. まとめ記事を固定にピン留め
2. マガジンでカテゴリ分け
3. タグは3〜5個つける
4. 過去記事を更新すると再浮上する可能性あり
