# XPシステム & レベル設計 ルールブック

## 概要

2つの完全に独立したシステムで構成される：

1. **XPシステム** — 真面目な英語力の指標。科学的根拠に基づく。ランダム要素一切なし。
2. **ボーナスシステム** — パチンコ的遊び要素。XPとは完全に別通貨。中毒性担当。

---

## Part 1: XPシステム（経験値）

### 設計思想

XPは「本物の英語力の蓄積」を数値化したもの。
点数の根拠はすべて認知心理学・第二言語習得研究に基づく。

### 科学的根拠

**Testing Effect** (Roediger & Karpicke, 2006)
- 「思い出す」行為が記憶を指数的に強化する
- 1回目のリコールで忘却曲線が約3倍伸びる
- 繰り返すたびに効果は累乗的に増加
- → リコール回数（Level 0-3）は **3の累乗** でスケール

**Production Effect** (MacLeod et al., 2010)
- 声に出すと黙読比で記憶定着が15-20%向上
- ただし「リコールの延長」ではなく「別次元のスキル」
- → 録音ボーナスは **固定値**（指数カーブとは分離）

**Generation Effect** (Slamecka & Graf, 1978)
- 自分で情報を生成する行為は受動的学習より定着が強い
- 研究ノートを書く = 能動的な深層処理
- → 解説ボーナスも **固定値**

### XPが発生するタイミング

| アクション | 条件 | 獲得XP |
|-----------|------|--------|
| マスタリーを1段階上げる | 0→1, 1→2, 2→3 | そのレベルのXP (9, 27, 81) |
| マスタリーをリセット | 3→0, 6→0 | 0（XPは発生しない） |
| 初回録音を追加 | mastery≥3 + 録音なし→あり | 50 |
| 初回リンクを追加 | mastery≥3 + 録音あり + リンクなし→あり | 50 |
| CROWN宣言 | 特別操作 | 100 |

|--------------|---------|----------|

| レギュラー | 20回 | 300-600 |
| ヘビー | 40回 | 600-1,500 |
| グラインド | 60回+ | 1,500-3,000+ |

## Part 2: プレイヤーレベル（ポケモン式）

### 設計思想

累計XPでプレイヤーレベルが上がる。
序盤はサクサク（即座の達成感）、後半はじわじわ（長期モチベ）。

### レベル計算式

```
必要累計XP(Lv) = floor(13 × Lv^2.3)
```

### レベルアップ演出

- バーが**95%以上**になると脈動開始（ニアミス効果）
- 満タン到達 → 光のバースト + レベル番号ドカン表示
- **称号変更時**（例: 修行者→実践者）→ さらに派手な専用演出
- Lv 100到達 → 全画面虹色エフェクト + 特別サウンド

### ファーストタッチボーナス

その日の**最初のレベルアップ**のみ、XPが **2倍** になる。

- 毎日アプリを開く動機になる
- 1日1回だけだから経験値の意味は壊れない
- 表示: 「FIRST TOUCH ×2!」の専用エフェクト

## Part 3: ボーナスシステム（パチンコ取扱説明書）

### ご利用の前に

本システムは純粋な娯楽要素です。
XP（経験値）とは**完全に独立**しており、学習成果には一切影響しません。
ボーナスで獲得する「スパーク」は、見た目や演出を楽しむためのものです。

### 遊び方

フレーズのレベルアップを行うたびに、自動的に**1回抽選**が発生します。
抽選結果に応じて「スパーク（SP）」と「演出エフェクト」が付与されます。

### 確率テーブル（出玉表）

```
┌──────────────┬──────┬──────────┬──────────────────────────────┐
│ 等級         │ 確率 │ スパーク │ 演出                         │
├──────────────┼──────┼──────────┼──────────────────────────────┤
│ ハズレ       │ 78%  │   1 SP   │ なし                         │
│ BONUS        │ 12%  │   5 SP   │ 金キラ                       │
│ GREAT        │  5%  │  15 SP   │ 画面揺れ + パーティクル       │
│ SUPER        │  3%  │  30 SP   │ フルスクリーン爆発            │
│ MEGA         │ 1.5% │  77 SP   │ 虹色演出 + スローモーション    │
│ LEGENDARY    │ 0.5% │ 777 SP   │ 全画面占拠 + 専用アニメーション │
└──────────────┴──────┴──────────┴──────────────────────────────┘
```

※ 確率は均等乱数により決定。外部要因による変動なし。
※ 天井（救済）: 200回ハズレ連続で次回MEGA以上確定。

### 期待値

```
1回あたりの期待SP = 0.78×1 + 0.12×5 + 0.05×15 + 0.03×30 + 0.015×77 + 0.005×777
                   = 0.78 + 0.60 + 0.75 + 0.90 + 1.155 + 3.885
                   = 8.07 SP/回

20回タッチ/日 → 約161 SP/日
```

### スパークの使い道（予定）

| SP | 交換アイテム |
|----|------------|
| 100 | エフェクトカラー変更 |
| 500 | レベルアップサウンド変更 |
| 1,000 | 称号の色カスタマイズ |
| 5,000 | 限定エフェクトパック |
| 77,777 | ??? (隠し要素) |

### 安全にお楽しみいただくために

- スパークは学習成果に**一切影響しません**
- LEGENDARYが出なくても英語力は同じペースで成長します
- 1日のタッチ回数に上限はありません
- 楽しみながら、結果的にたくさん回す。それが目的です。

```sql
CREATE TABLE IF NOT EXISTS player_stats (
    id TEXT PRIMARY KEY DEFAULT 'default',

    total_touches INTEGER DEFAULT 0,
    legendary_count INTEGER DEFAULT 0,
    pity_counter INTEGER DEFAULT 0,

    updated_at TEXT DEFAULT (datetime('now'))
```

## 数値バランスの根拠まとめ

| 設計要素 | 根拠 |
|---------|------|
| リコールXPが3の累乗 | Testing Effect: リコールが記憶を指数的に強化 |
| 録音/リンクが固定50 | Production/Generation Effect: 別次元のスキル |
| CROWNが100 | マスター到達のマイルストーン報酬 |
| レベルカーブが n^2.3 | 序盤の即座達成感 + 後半の持続挑戦 |
| ファーストタッチ2倍 | 日次エンゲージメント促進（1日1回限定で価値を保持） |
| ボーナス確率 | Variable Ratio Reinforcement（変動比率強化）= 最も中毒性の高い強化スケジュール |
| 天井200回 | 過度なフラストレーション防止の安全弁 |
